下载镜像

docker pull docker.io/library/nginx:latest

  • docker.io：仓库地址，官方的可省略
  • library：命名空间（作者名），官方的可省略
  • latest：标签（版本号），省略表示下载最新版

docker images：列出所有下载过的docker镜像

docker rmi ID/镜像名字 ：删除镜像
