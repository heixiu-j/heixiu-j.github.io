记一次搭建博客全过程

是使用Hexo + Github搭建的，主要参考知乎上的这篇文章

使用 Hexo+GitHub 搭建个人免费博客教程（小白向） - 知乎

安装node.js和git（git之前安装配置过）

安装完成后，命令行输入 node -v、npm -v和 git --version检查是否出现版本号，如下表示没问题

![image-20260329153706651](../img/setup blog/image-20260329153706651.png)

连接github

  • 在命令行设置用户名和邮箱
  • 创建SSH密钥

输入ssh-keygen -t rsa -C "GitHub 邮箱"，一路回车

  • 在github上添加密钥

进入 C:\users\用户名\.ssh目录（勾选显示隐藏的项目），用记事本打开 id_rsa.pub并复制里面的内容

打开github，进入setting页面，点击左侧栏的SSH and GPG keys，创建 New SSH key.

Title 随便取个名字，粘贴复制的id_rsa.pub内容到 Key 中，点击 Add SSH key 完成添加。

  • 打开git bash，输入 ssh -T git@github.com出现“Are you sure……”，输入 yes 回车确认。

显示 “Hi xxx! You've successfully……” 即连接成功。

这里出了一个小问题，我的电脑用户名设置成了中文，导致无法识别，感觉改的话会很麻烦，找了很多方法，都解决不了；最后觉得中文名以后肯定还会出问题，还是把中文名改成英文名了

(注意搞一些更改操作的时候最好建立一个还原点，弄错了还可以重来)

参考Windows11中文用户名改英文用户名！解决软件报错_哔哩哔哩_bilibili

创建Github Pages仓库

在github主页添加 New repository

repsository name :用户名.github.io

这里也要注意，后面有一步出问题，找半天才发现仓库名差了个github.io后缀

添加READ.ME文件

点击create repository创建

安装hexo

新建一个文件夹用来存放hexo程序文件，打开该文件夹，右键打开git bash

  • 输入一键安装
  • 初始化和本地预览

部署Hexo到GitHub Pages

  • 安装hexo-deployer-git:
  • 修改_config.yml文件末尾的Deployment部分
  • 完成后运行 hexo d将网站上传到GitHub Pages

这时访问我们的 GitHub 域名 https://用户名.github.io 就可以看到 Hexo 网站啦！！！

每次写文章后都可以通过 hexo g和 hexo s先本地预览，查看没问题了之后再上传
