发表于2026-03-28|computer
记一次搭建博客全过程是使用Hexo + Github搭建的，主要参考知乎上的这篇文章 使用 Hexo+GitHub 搭建个人免费博客教程（小白向） - 知乎 安装node.js和git（git之前安装配置过）安装完成后，命令行输入 node -v、npm -v和 git --version检查是否出现版本号，如下表示没问题 连接github 在命令行设置用户名和邮箱 12git config --global user.name "GitHub 用户名"git config --global user.email "GitHub 邮箱" 创建SSH密钥 输入ssh-keygen -t rsa -C "GitHub 邮箱"，一路回车 在github上添加密钥 进入 C:\users\用户名\.ssh目录（勾选显示隐藏的项目），用记事本打开 id_rsa.pub并复制里面的内容 打开github，进入setting页面，点击左侧栏的SSH and GPG keys，创建 New SSH key.
